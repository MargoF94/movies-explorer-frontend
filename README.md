<h1 align="center">Movies Explorer</h1>
<h3 align="center">A movies search engine <br> that allows you to sign in and save movies you'd like to give a try</h3>


- 🌱 I’m currently learning **HTML, CSS, JavaScript, React, Node.js, Express.js**

- 👨‍💻 My project is available [here](https://yourlocalmovieexplorer.nomoredomains.xyz/). This website is my graduation project for Yandex.Practicum.

<p align="left">
<b>Main page</b> contains some basic information about this project, me and technologies I used to make it.<br>
After <b>Registration</b> you can search movies and add them to your <b>Saved Movies</b> list. They can also be removed from the list at any time.
</p>
